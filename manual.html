产品概述

PCT-100 化工控制实训装置简介

产品简介

PCT-100 是一款基于 ESP32-C3 的物联网化工控制实训平台，专为化工专业教学和技能培训设计。 该装置集成了温度采集、光照检测、继电器控制、MQTT通信等核心功能，提供真实工业场景的实践教学环境。

核心亮点

🔧
模块化设计
采用模块化架构，支持灵活组合配置，满足不同教学需求
📊
实时监控
集成多种传感器，实时采集温度、光照等工艺参数
💻
ESP32主控
采用 ESP32-C3 高性能微控制器，支持 WiFi 和蓝牙
🔗

支持 MQTT 协议，便于云端集成和远程控制

{
    serial_read_task();      // 串口命令读取
    key_task();              // 按键状态机
    sensor_task();           // 传感器采集
    led_task();              // LED控制

    mqtt_manager_task();     // MQTT通信
    mqtt_report_task();      // 状态上报
}

按键控制逻辑

KEY1 - 自锁总开关

• 打开：允许本地控制，系统进入自动模式
• 关闭：继电器立即断开，禁止所有控制

KEY2 - 模式切换

• 短按（< 1500ms）：手动模式下切换继电器状态（00→01→10→11→00）
• 长按（≥ 1500ms）：在自动模式和手动模式之间切换
• 自动模式下短按无效

自动控制逻辑

温度控制（风扇）

温度 ≥ 阈值 → 风扇开启
温度 ≤ 阈值 - 2°C → 风扇关闭（回差防止频繁切换）

光照控制（灯泡）

光照 ≤ 阈值 → 灯泡开启
光照 ≥ 阈值 + 10lux → 灯泡关闭（回差防止频繁切换）

安装调试

详细的安装步骤

⚠️ 安装前请确保已阅读并理解安全操作规程，断电操作时务必确认电源已切断。

安装步骤

  1. 1
    硬件准备
    检查 PCT-100 实训板、USB Type-C 数据线、5V 电源适配器、灯泡和风扇模块是否齐全。
  2. 2
    连接负载设备
    将灯泡连接到继电器输出1，风扇连接到继电器输出2，确保接线正确无误。
  3. 3
    电源连接
    使用 USB Type-C 数据线连接实训板和电源适配器，确保电源输出为 5V/2A。
  4. 4
    开发环境配置
    安装 Arduino IDE 或 PlatformIO，添加 ESP32 开发板支持，选择 ESP32C3 Dev Module。
  5. 5
    代码下载
    打开 PCT-100.ino 工程，选择正确的端口和开发板，点击上传按钮下载程序。
  6. 6
    串口调试
    打开串口监视器，波特率设置为 115200，查看启动信息，确认系统正常运行。

WiFi 配置

// 连接 WiFi
wifi_set ssid YOUR_WIFI_SSID

wifi_save
wifi_connect

// 配置 MQTT Broker
mqtt_set broker mqtt.example.com
mqtt_set port 1883
mqtt_set user username
mqtt_set pass password
mqtt_set devid PCT_100_001
mqtt_save

维护保养

定期维护要点和故障处理

定期维护

🔧 每日检查
  • 检查电源指示灯是否正常
  • 检查传感器数值是否正常
  • 检查继电器工作状态
  • 清理设备表面灰尘
📅 每周维护
  • 检查电气连接是否紧固
  • 校准温度传感器零点
  • 检查执行机构动作是否正常
  • 备份程序和配置
🔄 每月保养
  • 检查所有传感器精度
  • 清洁光敏传感器表面
  • 检查继电器触点状态
  • 更新固件版本

常见故障处理

故障现象 可能原因 处理方法
温度显示异常传感器接线松动或损坏检查接线，更换传感器
光照无变化光敏电阻遮挡或ADC故障清理遮挡物，检查ADC通道
继电器不动作继电器故障或线路断路检查继电器，修复线路
WiFi 无法连接SSID/密码错误或信号弱重新配置WiFi，靠近路由器
MQTT 连接失败Broker地址错误或网络不通检查Broker配置，测试网络连通性
