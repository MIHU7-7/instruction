协议概述

PCT-100 的 MQTT 协议规范

协议信息

MQTT 协议参数
协议版本
MQTT 3.1.1
默认端口
1883 (TCP)
Keep Alive
30 秒
QoS 级别
QoS 0

主题结构

chemctrl/{device_id}/{topic}
chemctrl/{device_id}/status
设备状态上报（发布）
chemctrl/{device_id}/command
远程命令接收（订阅）

命令接口

MQTT 命令接口列表

POST set_relay
chemctrl/{device_id}/command
设置继电器状态
relay_num number - 继电器编号（3=灯泡，4=风扇）
value boolean - 状态（true=开启，false=关闭）
{ "cmd": "set_relay", "params": { "relay_num": 3, "value": true } }
POST set_mode
chemctrl/{device_id}/command
设置工作模式
mode string - 模式（auto=自动，manual=手动）
{ "cmd": "set_mode", "params": { "mode": "auto" } }
POST get_status
chemctrl/{device_id}/command
主动获取设备状态
{ "cmd": "get_status", "params": {} }
POST set_threshold
chemctrl/{device_id}/command
设置控制阈值
temp number - 温度阈值（°C）
light number - 光照阈值（lux）
{ "cmd": "set_threshold", "params": { "temp": 30.0, "light": 114.0 } }
POST reboot
chemctrl/{device_id}/command
重启设备
{ "cmd": "reboot", "params": {} }

状态上报

设备状态上报格式

PUBLISH Status Report
chemctrl/{device_id}/status
设备周期性上报状态信息（默认每5秒）
{ "device_id": "PCT_100_520", "temperature": 25.5, "light": 750, "mode": "auto", "key1": true, "lamp": false, "fan": true, "temp_threshold": 30.0, "light_threshold": 114.0, "timestamp": 1699999999 }
字段说明
device_id string - 设备唯一标识
temperature number - 当前温度（°C）
light number - 当前光照强度（lux）
mode string - 当前工作模式
key1 boolean - KEY1 状态
lamp boolean - 灯泡继电器状态
fan boolean - 风扇继电器状态
temp_threshold number - 温度控制阈值
light_threshold number - 光照控制阈值
timestamp number - 时间戳

串口命令

通过串口配置和控制设备

WiFi 命令

wifi_set ssid YOUR_WIFI_SSID # 设置 WiFi SSID wifi_set pass YOUR_WIFI_PASSWORD # 设置 WiFi 密码 wifi_save # 保存 WiFi 配置 wifi_connect # 连接 WiFi wifi_scan # 扫描周边 WiFi

MQTT 命令

mqtt_set broker mqtt.example.com # 设置 MQTT Broker 地址 mqtt_set port 1883 # 设置 MQTT 端口 mqtt_set user username # 设置 MQTT 用户名 mqtt_set pass password # 设置 MQTT 密码 mqtt_set devid PCT_100_001 # 设置设备 ID mqtt_save # 保存 MQTT 配置 mqtt_connect # 连接 MQTT Broker

错误码

ERR_INVALID_CMD
命令格式错误
ERR_INVALID_PARAM
参数无效
WARN_NOT_ALLOWED
当前模式不允许此操作
OK
操作成功
